MindMum

Product

MindMum app is designed to assist expectant and new mothers with the emotional challenges that this time of life often brings.

Requirement

User Interface and Experience Design (New Product)

Design Stratergy & Contribution
  1. Define Branding for the product based on the target user group (Color theory, Mood boards)
  2. Validate them with the client and the user group
  3. Wireframe the experience flow of the product (Hand-drawn sketches, Balsamiq)
  4. Usability testing with the user group
  5. High fidelity Mock-ups, with Branding (Adobe Photoshop, Sketch App)
  6. Implement the UI Iteratively, on each sprint based on feature priority (HTML, SASS)
  7. Iterative Usability testing with the user group on each sprint

Fitz

Product

Fitz is a chatbot assist, which helps you set & manage goals, achieve any positive behaviour changes you wanted to make in your life.

Problem Statement

Enhance the User Interface and Experience design of the product at par with the current design trends. The target user group are students of 18 - 24 age group. Current Website

Project Proposal

Iterative Enhancements - Approach for an existing product with a customer base

User test - User testing with the target user group at every phase

  1. Figure out the user pain points, feature wise
  2. Wireframe the new experience flow of the product resolving the pain points (Hand-drawn sketches, Balsamiq)
  3. High fidelity Mock-ups, with Branding (Adobe Photoshop, Sketch App)
  4. User test the new look and feel of the product with the target user group
  5. Workable prototype (HTML, SASS)
  6. Second level of user testing with the working prototype
Output

Positive response from the students on the new design.
